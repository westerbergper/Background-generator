git bash

Klona ett repository
git clone https://github.com/westerbergper/Background-generator.git


Kolla status 
git status

Lägga upp en fil
git add <file>

För att commita filen
git commit -m "Kommentar" 

För att lägga upp filerna
git push 


För att hämta nya ändringar
git pull

kolla branch
git branch

lägg till en ny branch
git branch littlefeature

för att byra branch
git checkout 

för att merga ihop en branch med master för att inte sakna nått som är gjort i master
git merge master
